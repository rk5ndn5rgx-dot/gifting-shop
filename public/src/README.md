# Baby Ray Studio 4.0 Source Layout

This folder contains the new studio architecture for the Baby Ray Studio upgrade.

Structure:
- `camera/` — full-screen camera core, mode switching, and trigger border logic
- `overlays/` — chat, gift, mood, council, enterprise, and ghost overlays
- `gifting/` — gift shop UI, Firestore integration, and gift animation helpers
- `council/` — zodiac ring, sigil animations, and council stage logic
- `enterprise/` — AI Enterprise dashboard, panel controls, and knowledge UI
- `ghost/` — Pisces Ghost mode, aura effects, and spirit overlays
- `scenes/` — avatar/skit video manager and scene switching
- `ui/` — reusable buttons, panels, sliders, and layout utilities
- `utils/` — shared helpers and Firebase connector
